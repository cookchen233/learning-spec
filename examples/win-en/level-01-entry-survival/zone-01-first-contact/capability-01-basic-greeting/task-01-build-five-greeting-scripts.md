## Task Goal

- 写出 5 个最小问候开场脚本，覆盖不同人物、地点与目的。

## Input

- capability 的 `Scenario`
- capability 的 `Success Path`

## Examples

- Example 1:
  - Hello. Excuse me, I need some help with check-in. Thank you.
- Example 2:
  - Hi. Excuse me, I want to ask about the menu. Thank you.

## Exercises

- Exercise 1:
  - Person: shop assistant
  - Place: store
  - Purpose: find an item
  - Instruction:
    - 先自行写出脚本，再与参考答案对照。
  - Reference Answer:
    - Hello. Excuse me, I need help finding this item. Thank you.

- Exercise 2:
  - Person: landlord or agent
  - Place: apartment viewing
  - Purpose: ask about a room
  - Instruction:
    - 先自行写出脚本，再与参考答案对照。
  - Reference Answer:
    - Hi. Excuse me, I want to ask about this room. Thank you.

- Exercise 3:
  - Person: school or office staff
  - Place: front desk
  - Purpose: ask about a form

- Exercise 4:
  - Person: airport staff
  - Place: airport gate area
  - Purpose: ask about boarding

- Exercise 5:
  - Person: hotel staff
  - Place: hotel reception
  - Purpose: ask about check-in

## Action

- 先读懂 2 个参考例子中的结构。
- 完成 Exercise 1 到 Exercise 5，5 题都要先自行写出一个 10-20 秒最小开场脚本。
- Exercise 1 和 Exercise 2 只允许在自行作答后再看参考答案。
- 每个脚本只保留问候、礼貌意图、结束。
- 明确标注每个脚本里的三类核心元素。

## Proof

- 针对 5 个 exercises 完成 5 个不同场景的脚本文本。
- 每个脚本都对应人物、地点、目的三类元素。

## Done Condition

- [ ] 针对 5 个 exercises 的脚本已写出
- [ ] 每个脚本的三类元素已明确
